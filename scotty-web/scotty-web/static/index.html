Development Roadmap

Urban mobility shifts form a composable monoid of higher-order functions, enabling type-safe, reproducible sensitivity analysis of transport emissions across policy scenarios — a formulation absent from existing transport modeling literature.

WEEKS 1–2
Core Domain Model

  • TransportMode ADT
  • MobilityProfile
  • EmissionFactors
  • Validation layer
WEEKS 2–3
Shift Algebra

  • Primitive shifts
  • Monoid composition
  • Policy DSL
  • Conditional shifts
WEEKS 3–4
